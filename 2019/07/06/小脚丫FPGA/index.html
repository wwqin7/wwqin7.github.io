小脚丫FPGA

发表于 2019-07-06 更新于 2019-07-07 阅读次数：

FPGA

在做实验之前，可以先在小脚丫PFGA学习交流平台初步了解FPGA，这个STEP FPGA开源社区，有很多资料可供参考学习。

实验室的STEP-MAX10-08SCM小脚丫在STEP-MAX10有详细介绍，可以先了解一下它的硬件结构、看看引脚对照表。后边的实验涉及引脚的选择，注意不要选错！另外，该网页下面的pdf文档资料很重要，有助于我们进行实验，所以一定要看看。

FPGA硬件编程语言主要有Verilog HDL和VHDL两种，建议学习Verilog，可先了解基本语法方便后面进行实验。

Quartus II软件安装及使用

首先，在 Intel-Altera 官网 下载 Quartus II 软件，但这个下载时间较长，建议直接拿实验室的u盘将安装包拷到电脑上。
之后按照 step-max10_v2软件手册v1.0 进行安装，并创建项目完成一个LED灯的基础实验。

注意事项：
新建工程时需要注意FPGA器件的选择，软件手册中是以 10M02SCM153I7G 的型号为例，但实验室的小脚丫为 10M08SCM153I7G ，所以这一步应根据实际器件为准进行选择。否则，后续的FPGA加载过程可能会失败。

FGPA烧录时，需要确认硬件驱动USB-Blaster[USB-0]，若找不到该硬件，可能是驱动未安装。此时，可进入软件安装目录找到 quartus\drivers文件夹中的应用程序进行安装。

按照软件手册做仿真实验，并未成功实现仿真，网上找了一些解决办法也都失败了…但这没有关系，实验中我们直接使用示波器显示信号波形。但有兴趣的话可以试试仿真，看能不能成功。

两个小实验

1.分频与倍频

调用 Quartus II 软件中的PLL IP核，可以轻松实现分频、倍频，完成多个不同频率信号的输出。

实现方法

同样地，先建立一个新的工程，之后找到 Tools>>IP catalog 中的“ALTPLL”，具体调用过程可以参考网页 QuartusII调用PLL IP核实现分频、倍频 。

顶层文件IP_PLL.v示例：

module IP_PLL//ip核PLL实现倍频、分频
(
  CLK,LED1,LED2
);
input CLK;
output LED1;//倍频25M
output LED2;//分频1M

PLL    PLL_inst //复制的实例模板
(
.inclk0 ( CLK ),
.c0 ( LED1 ),
.c1 ( LED2 )
);

endmodule

注意：复制实例模版PLL_inst.v文件时，可能需要修改括号中的参数名称，因为实例模板括号中的参数应该和顶层文件的参数保持一致。

另外，网页教程第15步后的仿真操作可以忽略，我们直接使用示波器显示信号波形。完成“Analysis&Synthesis”后，再进行管脚约束。可自行分配输出管脚，但在使用示波器探头检测信号时切勿接错管脚。

2.移频

在无源反向散射通信系统中，我们需要对待传输信息进行处理，如调制相关信号，而在频移键控调制的过程中就需要进行移频操作，其原理图如下：

实验仅进行是原理中的移频操作，即Step1, 10KHZ 和 10MHZ 的信号分别对应Signal和f1，经处理后产生Step1的移频后输出信号。

实现方法

小脚丫FPGA MAX10外部时钟频率为12MHZ。该实验先调用 Quartus II 软件中的PLL IP核分频产生 10KHZ 和 10MHZ 两个信号。

顶层文件backscatter.v如下：

module backscatter (clk,clkout);//fpga的12MHZ信号经PLL分频产生10MHZ、10KHZ信号,进行移频

    input     clk;                       //输入信号，其中clk连接到FPGA的C1脚，频率为12MHz
    output    clkout;                    //输出信号，可以连接到LED观察分频的时钟
      wire[1:0] temp1;
      wire[1:0] temp2;

 PLL m1(.inclk0(clk),
            .c0(temp1),//10M
            .c1(temp2)//10K    
            );
    process m2(
    .clk1(temp1),
    .clk2(temp2),
    .clkoutp(clkout)
    );

endmodule

之后编写一个中间处理文件process.v对这两个信号进行处理，最终产生一个经移频后的信号，可用示波器观察波形。

module process ( clk1,clk2,clkoutp );

    input     clk1,clk2;                       //输入信号，其中clk连接到FPGA的C1脚，频率为12MHz
    output    clkoutp;            //输出信号，可以连接到LED观察分频的时钟  

    assign clkoutp=clk1&clk2;

endmodule
